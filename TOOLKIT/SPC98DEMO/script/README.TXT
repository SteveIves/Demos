____________
Introduction

This example (scrcmp.dbr) shows how the external Script interface 
is used.  Refer to chapter 2, "Script," in the UI Toolkit Manual 
for a description of the external Script interface routines.

________________________
Building the application

Compile scrcmp.dbl and link with WND:tklib.elb (TKLIB_SH.EXE on VMS),
RPSLIB:ddlib.elb, and WND:script.elb (SCRIPT_SH.EXE on VMS).  On 
Windows, you can run scrcmp.bat to do this.  On UNIX and VMS, you can 
run the scrcmp batch file.
