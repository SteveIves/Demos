WebGen Re-Licencing

On this floppy disk you will find a program called relicence.exe.  This program will reset your WebGen evaluation licence count to 20.  This copy of the relicence program has the following restrictions:

  • It must be executed a floppy disk in drive A:
  • It will only execute until (and including) Friday 26th November 1999.
  • It will only execute once on a given system.

To relicence  your WebGen evaluation software, double click on the relicence.exe program on drive A: in Windows Explorer, or select "Start / Run" and type in A:\relicence.exe , then press OK.

Once the patch has been applied you may discard this disk, it is useless!
